2 TERMUX 及EMACS安装

TERMUX在应用商店或者网上搜索，很容易找到，安装和普通程序安装一致。安装完成后按提示 pkg update & pkg upgrade ，接下来就像UBUNTU中一样，pkg install emacs,而EMACS的配置可以参照陈斌redguardtoo一年成为EMACS高手 。

3 环境配置

3.1 Termux 键盘设置

当用触摸屏的软键盘时发现可以输入中文，但是用实体键盘时，最开始死活输入不进去。可以在TERMUX左侧向右滑调出菜单，在左下KEYBOAED上长按可以调出CTRL和ALT等功能键，用软键盘时可以用此方法输入功能键。同时在功能键盘上向左滑可以调出一个输入框，在这个框中就可以输入中文了。

此方法输入中文真的很不方便，每一句中文除了要按一下空格，还要按一下回车键。这里顺便提一下，华为手机自带的百度输入法，是可能完美支持实体键盘的。有人讲实体键盘输入时总要弹出一个软键盘，占了很宽的屏，这是在输入法中没有设置好。在输入法中设置不显示软件盘即可。使用时有几个快捷键要记住：ALT+SPACE 调出输入法，按第一次是输入符，再按一次是正常输入。SHIFT+SPACE是中文和英文输入切换。SHIFT+ENTER是输入方法切换，即拼音、笔画、五笔等切换。只要记住这些快捷键，用实体键盘的体验比软键盘好了不知多少倍。

前面说到TERMUX中输入中文很是不便，但是EMACS可是自带中文拼音输入法，默认的切换方式是CTRL+\。另外如果不喜欢拼音输入法，还可能用EMACS五笔输入法 。配置方法请参考帮助说明，此处不再重复。但是如果你也像我一样用了Redguardtoo的配置文件，就要注意了，可能怎么也进不了五笔，调出的总是拼音。原因是他加载的init-chinese.el中将pyim设置为默认的输入法了，至于怎么改就看你心情了。

此时可能你会发现一个很大的问题，中英文的切换怎么也搞不好，即使在EMACS的启动配置中明确用 global-unset-key 取消了CTRL-SPACE到set-mark-command的绑定，又重新绑定到toggle-input-method，还是不行。经过测试，在TERMUX中，按CTRL+SPACE等同于CTRL+@，而按CTRL+SHIFT+2等同于SHIFT+2，即均是输入@符号。也就是说键盘映射不正确，到现在都没有找到解决的办法。但是，中英文切换的方法还是有的，就是在EMACS中用M-x执行global-unset-key 取消CTRL-SPACE就可以进行中文输入法的开启与关闭了。但牺牲了 set-mark-command。

如果想知道每个按键具体是怎么映射的，可以运行 vtshowkeys 进行查看。要注意的是，也可能是我的手机才存在这个问题，只是没有在其他手机上进行过测试。

3.2 PYTHON环境配置及TERMUX环境增强

在简书上有文章专门讲如何配置将EMACS配置为好用的PYTHON IDE环境 ，可以参照安装。这里讲一些增强TERMUX环境功能的经验。

3.2.1 增强WGET的功能

wget命令超级好用是不可否论的，Termux busybox自带此功能，但是遇到 https的链接就出问题了，因为它不支持 –no-check-certificate ,于是网上的办法都没有用了。如果你有代理服务器的话，倒是可以在服务器上下载了，再用scp命令复制到本地也是可以的。如果你不能忍受麻烦，倒是可能下载原代码编译安装。首先肯定要准备编译环境了，TERMUX会提醒你该安装哪个包，比如 gcc 就在 clang 包中。要点是 configure 时要设置安装目录，而/bin/sh也是找不到的，这是因为 TERMUX的目录结构与原生LINUX的不一样，修改相关路役即可。

3.2.2 Python pillow 安装

pillow 安装时依赖于许多其他的包，可能参照原生linux 下的安装方法，但是libzlib在termux的包管理库里没有，可能下载源码包进行安装。

3.2.3 Python lxml 安装

在用Python 解析 html文件时，可能会用到lxml，但他依赖的包中有一个是 libxml-dev,即使是安装了后，也可能出出现找不到的情况，同样是路径的问题，要在 /data/data/com.termux/files/usr/include 目录中使用 ln -s libxml2/libxml libxml 建立一个链接，保证编译程序能找到需要的头文件。
